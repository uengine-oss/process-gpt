# 프로세스 인스턴스 파일 정리 명세

## Purpose
시스템이 완료된 프로세스 인스턴스에 연결된 스토리지 첨부 파일을 주기적으로 정리하는 능력을 보장한다.

## Requirements

### Requirement: 완료 인스턴스 첨부 파일 정리
시스템은 완료 상태이며 아직 정리되지 않은 프로세스 인스턴스를 주기적으로 감지하여 해당 인스턴스에 연결된 스토리지 파일을 SHALL 삭제한다.

#### Scenario: 완료 인스턴스 파일 삭제
- **GIVEN** `status`가 `COMPLETED`이고 정리되지 않은(`is_clean_up=false`) 프로세스 인스턴스에 소스 파일이 연결되어 있다
- **WHEN** 파일 정리 주기(5분)가 도래한다
- **THEN** 시스템은 해당 인스턴스의 소스 파일 정보를 조회하여 스토리지의 파일을 삭제한다

#### Scenario: 정리 대상 없음
- **GIVEN** 완료되었고 정리되지 않은 프로세스 인스턴스가 없다
- **WHEN** 파일 정리 주기가 도래한다
- **THEN** 시스템은 삭제 작업 없이 다음 주기를 기다린다
