# CrewAI 액션 에이전트 도구 우선순위 명세

## Purpose
설정자가 에이전트의 작업 계획·실행 시 도구와 스킬이 고려되는 우선순위를 제어할 수 있도록 보장한다. 우선순위를 지정하지 않으면 스킬 보유 여부에 따른 일관된 기본 우선순위가 적용된다.

## Requirements

### Requirement: 사용자 지정 도구 우선순위 적용
시스템은 에이전트 설정에 `tool_priority_order`(또는 별칭 `tool_priority`) 리스트가 제공되면, 그 순서에 따라 에이전트의 도구·스킬 우선순위를 SHALL 적용한다. 리스트에서 앞쪽 항목일수록 높은 우선순위를 가지며, 항목은 스킬명, 스킬 MCP 서버명(`claude-skills`, `computer-use`), 도구명(`dmn_rule`, `mem0`, `memento` 등)을 혼합할 수 있다.

#### Scenario: 지정한 우선순위 순서 반영
- **GIVEN** 에이전트 설정에 `tool_priority_order`로 도구 우선순위가 지정되어 있다
- **WHEN** 시스템이 에이전트의 도구 목록을 구성한다
- **THEN** 도구는 지정된 우선순위 순서대로 정렬된다
- **AND** 동일 우선순위 항목 간에는 원래 노출 순서가 보존된다

#### Scenario: 스킬명을 통한 우선순위 지정
- **GIVEN** `tool_priority_order`에 에이전트에 할당된 스킬명이 포함되어 있다
- **WHEN** 시스템이 도구 우선순위를 적용한다
- **THEN** 해당 스킬명은 스킬 MCP 서버(`claude-skills`, `computer-use`)의 우선순위 위치로 반영된다

### Requirement: 와일드카드를 통한 미기재 도구 위치 지정
시스템은 `tool_priority_order`에 와일드카드 `*`가 포함되면, 우선순위 리스트에 명시되지 않은 나머지 도구를 `*` 위치에 SHALL 묶어 배치한다. `*`가 생략되면 미기재 도구는 가장 낮은 우선순위로 배치한다.

#### Scenario: 와일드카드 위치에 미기재 도구 묶음
- **GIVEN** `tool_priority_order`에 일부 도구명과 `*`가 함께 지정되어 있다
- **WHEN** 시스템이 도구 우선순위를 적용한다
- **THEN** 명시되지 않은 도구들은 `*`가 위치한 우선순위 자리에 함께 배치된다

#### Scenario: 와일드카드 생략 시 미기재 도구 후순위 배치
- **GIVEN** `tool_priority_order`에 `*`가 포함되지 않았다
- **WHEN** 시스템이 도구 우선순위를 적용한다
- **THEN** 명시되지 않은 도구들은 가장 낮은 우선순위로 배치된다

### Requirement: 기본 도구 우선순위 적용
시스템은 에이전트 설정에 사용자 지정 우선순위가 없으면, 스킬 보유 여부에 따른 기본 우선순위를 SHALL 적용한다. 스킬을 보유한 에이전트는 스킬 MCP 도구(`claude-skills`, `computer-use`)를 최우선으로 두고 `dmn_rule`, `mem0`, 기타 도구 순으로 정렬하며, 스킬이 없는 에이전트는 `dmn_rule`, `mem0`, 기타 도구 순으로 정렬한다.

#### Scenario: 스킬 보유 에이전트의 기본 우선순위
- **GIVEN** 에이전트가 스킬을 보유하고 사용자 지정 우선순위가 없다
- **WHEN** 시스템이 도구 우선순위를 적용한다
- **THEN** 스킬 MCP 도구가 최우선으로 정렬되고 이어서 `dmn_rule`, `mem0`, 기타 도구 순으로 정렬된다

#### Scenario: 스킬 미보유 에이전트의 기본 우선순위
- **GIVEN** 에이전트가 스킬을 보유하지 않고 사용자 지정 우선순위가 없다
- **WHEN** 시스템이 도구 우선순위를 적용한다
- **THEN** 도구는 `dmn_rule`, `mem0`, 기타 도구 순으로 정렬된다

### Requirement: 매니저 에이전트 우선순위 해소
시스템은 크루의 매니저 에이전트에 적용할 도구 우선순위를 호출 시 전달된 우선순위, 첫 에이전트 설정의 우선순위, 기본 우선순위 순으로 SHALL 해소한다. 해소된 우선순위는 생성되는 작업 지시문의 도구 우선순위 안내에 반영된다.

#### Scenario: 우선순위 해소 우선 적용 순서
- **GIVEN** 호출 인자로 전달된 우선순위와 첫 에이전트 설정의 우선순위가 모두 존재한다
- **WHEN** 시스템이 매니저 에이전트의 우선순위를 결정한다
- **THEN** 호출 인자로 전달된 우선순위가 우선 적용된다

#### Scenario: 우선순위 미지정 시 기본값 적용
- **GIVEN** 호출 인자와 첫 에이전트 설정 모두 우선순위를 지정하지 않았다
- **WHEN** 시스템이 매니저 에이전트의 우선순위를 결정한다
- **THEN** 스킬 보유 여부에 따른 기본 우선순위가 적용된다
- **AND** 해소된 우선순위가 작업 지시문의 도구 우선순위 안내에 반영된다
