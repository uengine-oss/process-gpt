- [ ] 2.1 현재 분석 대시보드의 위젯 목록을 안정적 `widgetId` 키로 정의한 레지스트리 모듈 작성 (제목, 렌더 컴포넌트, 기본 `{x,y,w,h}`, `minW/minH`, 기본 표시 여부)
- [ ] 2.2 기존 `Dashboard.vue` 의 각 차트/표/카드를 레지스트리 컴포넌트로 분리·매핑
